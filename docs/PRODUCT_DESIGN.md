## 一、产品概述

### 1.1 产品愿景

打造一个面向影视创作者的 **AI 短片制作平台**，让导演、编剧、内容创作者能够通过 AI 技术，快速将创意转化为可视化的短片作品。

### 1.2 目标用户

| 用户类型 | 描述 | 核心需求 |
|---------|------|---------|
| **独立导演** | 独立电影/短片制作者 | 快速制作概念片、预览片、分镜动画 |
| **内容创作者** | 短视频博主、自媒体 | 批量生成高质量视频内容 |
| **广告从业者** | 广告公司创意人员 | 快速出方案、制作广告 Demo |
| **影视学生** | 电影学院学生 | 学习与实践影视创作 |
| **企业用户** | 品牌营销团队 | 制作企业宣传片、产品视频 |

### 1.3 核心价值主张

```
"从创意到成片，AI 全程助力"

1. 🎨 一站式创作：生图 → 生视频 → 剪辑 → 合成 → 导出
2. 🔗 工作流串联：素材交叉引用，任务依赖管理
3. 🎬 专业输出：符合影视行业标准的项目管理
4. ⚡ 高效迭代：快速尝试不同创意方向
5. 💰 成本可控：按量计费，预算透明
```

### 2.2 核心功能详解

#### 2.2.1 🎨 AI 生图模块

**功能描述**：通过文字描述或参考图片生成高质量图片

| 子功能 | 说明 | 参数 |
|-------|------|-----|
| **文生图** | 输入提示词生成图片 | 提示词、负向提示词、分辨率、宽高比、风格 |
| **图生图** | 基于参考图生成新图片 | 参考图、强度、提示词 |
| **图片搜索** | 搜索参考素材 | 关键词、风格筛选 |

**技术实现**：
- 调用 `gemini-image-gen` 技能
- 模型：DMXAPI Gemini 3.1 Flash Image Preview
- 支持 0.5K/1K/2K/4K 分辨率
- 支持 14 种宽高比

**交互流程**：
```
1. 用户输入提示词（支持中英文）
2. 可选：上传参考图
3. 选择分辨率、宽高比、风格
4. 点击生成 → 显示进度 → 展示结果
5. 结果操作：下载、收藏、编辑、用于生成视频
```

#### 2.2.2 🎬 AI 生视频模块

**功能描述**：通过多种方式生成视频内容

| 子功能 | 说明 | 参数 |
|-------|------|-----|
| **文生视频** | 文字描述生成视频 | 提示词、分辨率、时长、是否生成音频 |
| **图生视频** | 图片动画化为视频 | 首帧图片、提示词、分辨率、时长 |
| **首尾帧生视频** | 指定首尾帧生成过渡视频 | 首帧图片、尾帧图片、提示词、时长 |

**技术实现**：
- 调用 `seedance-text2video`、`seedance-image2video`、`seedance-frame2video` 技能
- 模型：Doubao Seedance 1.5 Pro
- 支持 480p/720p/1080p
- 时长 4-12 秒
- 原生音画同步

**交互流程**：
```
1. 选择生成方式（文/图/首尾帧）
2. 输入提示词 / 上传图片
3. 选择分辨率、时长、是否生成音频
4. 点击生成 → 异步任务 → 通知完成
5. 结果操作：预览、下载、收藏、添加到时间线
```

#### 2.2.3 🎵 AI 音乐生成模块（NEW！）

**功能描述**：AI 生成背景音乐和歌曲

| 子功能 | 说明 | 参数 |
|-------|------|-----|
| **文生音乐** | 描述风格，AI 自动生成歌词和旋律 | 提示词、时长、模型 |
| **自定义歌词** | 用户提供歌词，AI 谱曲演唱 | 歌词、风格、人声性别、时长 |
| **纯音乐** | 生成无歌词的背景音乐 | 提示词、风格、时长 |

**技术实现**：
- 调用 `evolink-music` 技能
- 模型：Suno v4 / v4.5 / v5
- 支持 30-240 秒时长
- 80+ 风格标签

**参数详解**：

| 参数 | 说明 | 选项 |
|-----|------|-----|
| **模式** | 简单/自定义 | simple: AI写词 / custom: 用户写词 |
| **人声** | 是否包含人声 | instrumental: 纯音乐 / vocal: 带人声 |
| **风格** | 音乐风格标签 | pop, rock, jazz, electronic, lo-fi, cinematic 等 |
| **情绪** | 情感基调 | happy, sad, romantic, energetic, calm 等 |
| **人声性别** | 演唱者性别 | male / female |
| **时长** | 目标时长 | 30-240 秒 |
| **模型** | 生成模型 | suno-v4 (平衡) / v4.5 (风格控制) / v5 (工作室级) |

**交互流程**：
```
1. 选择生成模式（简单/自定义/纯音乐）
2. 输入音乐描述或歌词
3. 选择风格、情绪、时长等参数
4. 点击生成 → 异步任务（约60-120秒）→ 通知完成
5. 结果操作：试听、下载、收藏、添加到项目音轨
```

**典型场景**：
```
场景1：为视频配背景音乐
- 选择"纯音乐"
- 描述："紧张刺激的动作场景配乐，快节奏，鼓点强烈"
- 选择时长：30秒
- 生成后添加到时间线音轨

场景2：创作主题曲
- 选择"自定义歌词"
- 输入歌词：[Verse] ... [Chorus] ...
- 选择风格："pop, ballad, emotional"
- 选择人声："female"
- 生成完整歌曲
```

#### 2.2.4 ✂️ 图片编辑模块

**功能描述**：对生成的图片进行后期处理

| 编辑功能 | 说明 |
|---------|------|
| **裁剪/调整尺寸** | 自定义裁剪区域和输出尺寸 |
| **滤镜** | 预设滤镜效果（电影感、复古等） |
| **调色** | 亮度、对比度、饱和度、色温 |
| **局部重绘** | 选择区域重新生成 |
| **扩展画布** | AI 扩展图片边界 |
| **去水印/修复** | 移除不需要的元素 |

#### 2.2.4 🎞️ 视频编辑模块

**功能描述**：对生成的视频进行剪辑和后期处理

| 编辑功能 | 说明 |
|---------|------|
| **裁剪** | 截取视频片段 |
| **拼接** | 多个视频按顺序连接 |
| **转场** | 添加淡入淡出等转场效果 |
| **调速** | 快放/慢放 |
| **添加音频** | 上传背景音乐或使用 AI 生成 |
| **字幕** | 添加文字字幕 |
| **滤镜** | 色彩调整和风格滤镜 |

**功能描述**：将多个生成任务串联成完整的工作流


**核心概念**：

| 概念 | 说明 |
|-----|------|
| **节点 (Node)** | 单个生成/编辑任务 |
| **连接 (Edge)** | 节点之间的数据流向 |
| **工作流 (Workflow)** | 完整的任务流程图 |
| **模板 (Template)** | 可复用的工作流模板 |

**支持的操作**：
- 可视化拖拽编辑
- 节点参数配置
- 条件分支（根据上一步结果选择不同路径）
- 并行执行（多个节点同时运行）
- 断点续传（任务失败后从断点继续）

```prisma
// 用户
model User {
  id            String    @id @default(cuid())
  email         String    @unique
  phone         String?   @unique
  password      String
  nickname      String

  role          UserRole  @default(USER)

model Quota {

  updatedAt       DateTime  @updatedAt

// 项目
model Project {
  id          String      @id @default(cuid())

  description String?
  cover       String?     // 封面图
  status      ProjectStatus @default(DRAFT)
  workflow    Json?       // 工作流定义
  timeline    Json?       // 时间线数据
  assets      Asset[]
  tasks       Task[]
  exports     Export[]
  createdAt   DateTime    @default(now())
  updatedAt   DateTime    @updatedAt
}

// 素材
model Asset {
  id          String      @id @default(cuid())
  userId      String
  user        User        @relation(fields: [userId], references: [id])
  projectId   String?
  project     Project?    @relation(fields: [projectId], references: [id])
  type        AssetType   // IMAGE, VIDEO, AUDIO
  name        String
  url         String      // OSS URL
  thumbnail   String?     // 缩略图
  metadata    Json?       // 宽高、时长等元信息
  source      AssetSource // GENERATE, UPLOAD, EDIT
  taskId      String?
  task        Task?       @relation(fields: [taskId], references: [id])
  createdAt   DateTime    @default(now())
  updatedAt   DateTime    @updatedAt
}

// 任务
model Task {
  id          String      @id @default(cuid())
  userId      String
  user        User        @relation(fields: [userId], references: [id])
  projectId   String?
  project     Project?    @relation(fields: [projectId], references: [id])
  type        TaskType    // IMAGE_GEN, VIDEO_GEN, VIDEO_EDIT, etc.
  status      TaskStatus  @default(PENDING)
  params      Json        // 任务参数
  result      Json?       // 任务结果
  error       String?
  asset       Asset?
  createdAt   DateTime    @default(now())
  updatedAt   DateTime    @updatedAt
  startedAt   DateTime?
  completedAt DateTime?
}

// 导出记录
model Export {
  id          String      @id @default(cuid())
  projectId   String
  project     Project     @relation(fields: [projectId], references: [id])
  format      String      // mp4, webm, gif
  resolution  String      // 1080p, 720p
  url         String?     // 导出文件 URL
  status      ExportStatus @default(PENDING)
  createdAt   DateTime    @default(now())
}
```
